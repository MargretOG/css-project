Difference between paddings, margings and border

PADDING: This is the inner space between the content and the border. it control how much whitespace exists within a box, there can be set from zero and above. nagative values don't work for padding.

MARGINS: is the outer space surrounding the box. it define how element interacts with the edge of the screen. margins can be set as auto,zero or negative values

BORDER: it goes around the padding and content
